# 🎲 Dice Game (Python)

A simple Python dice game where the user plays against the computer.

## 📌 How it works
- User and computer both roll a dice (1 to 6)
- Higher number wins the round
- Game has 3 rounds
- Final score decides the winner

## ▶️ How to run
1. Open terminal
2. Run the file:
   
   Roll dice game.py
## 📖 Options in game
- Start Game
- Read Instructions
- Exit

## 🛠️ Built with
- Python 3.13.13
- Random module(random.randint)
